Add Google Adsense ads using widget or shortcode. Supports up-to-date ad formats, responsive sizing and adsense placeholders!

== Description ==

Adsense made easy!

  1. Add widget or shortcode on your site. 
  2. Select ad format, make it responsive.
  3. Watch your earnings rocket ;) 
  
*Features*

  1. Up-to-date Ad Formats 
  2. Easy-to-use Widget
  3. Super-easy [shortcode](https://metinsaylan.com/wordpress/plugins/adsense-widget/shortcode/)
  4. Ultimate options panel
  5. Auto insert post ads
  6. Responsive [ad sizing](https://metinsaylan.com/wordpress/plugins/adsense-widget/sizing-options/)

Shortcode usage:

`[adsense userid="pub-xoxox" format="display" align="none" slot="xxx" sizes="sm-banner md-rectangle lg-leaderboard"]`

Or you can setup the default options and just use:

`[adsense]`

== Installation ==

1. Install and activate the plugin from `Plugins > Add New > Search`
2. After activation go to `Settings > Adsense Widget` 
3. Add your `Publisher Id`
4. Choose default `Shortcode` settings and Save
5. Add widget under `Appearance > Widgets` or
6. Use shortcode to add `[adsense]` units to your post
